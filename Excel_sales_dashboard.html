Sample Sales Data Dashboard [Excel]

A dynamic dashboard showcasing sales data of different vehicles over three years using Excel. PivotTables and PivotCharts are utilized to easily display the data.

PROGRAM USED: Excel
SKILLS USED: #Visualization, #Dashboards
MAIN TOOLS USED: PivotTable, PivotChart (with slicers)

PART 1: DATA SOURCE

The data is from kyanyoga's Kaggle dataset. It contains an Excel worksheet of fictitious retail sales data of various vehicles. Information such as quantity sold, total revenue from each order, and customer information is included.

PART 2: PIVOTTABLES & PIVOTCHARTS

PivotTables and PivotCharts are then created to display various relationships between the data:

  • Sum of Sales by Country and Vehicle Type (PRODUCTLINE in dataset) - Dynamically displays the total sales for each country, broken down further by vehicle type. A stacked bar chart is created.
  • Sum of Sales by Vehicle Type and Year (YEAR_ID in dataset) - Dynamically displays the total sales for each vehicle type per year. A clustered column chart is created, with each year its own column.
  • Quantity Ordered by Year and Vehicle Type - Dynamically displays the total quantity ordered of each vehicle type per year. A line chart with markers is created, showing the changes in how many of each were ordered over time.
  • Quantity Ordered by Year and Vehicle - Statically displays total quantity ordered of each vehicle type per year. Pie charts are created for each year to show the overall distribution.

The PivotTables are sorted based on desired order. For example, Sum of Sales by Country and Vehicle Type is ordered by country's grand total of sales, descending.

PART 3: DASHBOARD

A dashboard is created on a new worksheet; the layout is designed using shapes and built-in Excel themes. The PivotCharts are copied onto this dashboard and formatted accordingly—field buttons are removed, axes are reformatted, titles are added.

Slicers are used to connect dynamically with the three main charts. From here, information can be easily filtered all from the dashboard. For example, selecting “Classic Cars” will change the view on all three main charts to automatically display only information pertaining to “Classic Cars.”

Information can be added to the main table afterwards and refreshed to quickly and efficiently display visualizations, making the interactive dashboard a useful feature.
