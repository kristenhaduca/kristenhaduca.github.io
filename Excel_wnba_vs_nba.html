WNBA vs. NBA Salaries [EXCEL]

A comparison of salaries and game statistics of players in the WNBA and the NBA using Excel. PivotTables, PivotCharts, and column charts are used to compare the two leagues' salaries across different player attributes.

PROGRAM USED: Excel
SKILLS USED: #Power Query, #Data Cleaning, #Data Wrangling, #Central Tendency & Dispersion, #Visualization

MAIN FORMULAS USED: INDEX(), MATCH(), IF(), ISERROR(), AGGREGATE(), AVERAGE(), MEDIAN(), MAX(), MIN(), MODE(), VAR.S(), STDEV.S(), SUMIFS(), COUNTIFS(), QUARTILE.INC(), AVERAGEIF()

PART 1: DATA SOURCE

The following centers around getting data for NBA players. WNBA players' data has already been compiled and cleaned in WNBA Salaries & Statistics [EXCEL].

PART 2: DATA CLEANING & WRANGLING

Main NBA Table:

All information gets combined into one main NBA table using INDEX() and MATCH(). First, the tables with the salaries data are combined. ISERROR() is nested within an IF() function to return a blank cell if there is an #N/A error after the INDEX()/MATCH()—this error being caused by no salary information being listed because of things like the player being in college or having not played in the season.

=IF(ISERROR(INDEX('NBA 19-20'!$D$2:$D$529,MATCH('NBA 20-21'!$B2,'NBA 19-20'!$B$2:$B$529,0))),"",INDEX('NBA 19-20'!$D$2:$D$529,MATCH('NBA 20-21'!$B2,'NBA 19-20'!$B$2:$B$529,0)))

PowerQuery is used to separate the player names from their position using a comma and space (", ") as the delimiter. Any blank columns within the main table are deleted as well. Salary amounts are changed to the currency format.

Data Tab > Get Data From Table/Range > Split Column by Delimeter: ", "
= Table.TransformColumnTypes(#"Split Column by Delimiter",{{"Player.1", type text}, {"Player.2", type text}})

Game statistics and draft year are appended to the main table from their own respective tables using INDEX() and MATCH(). Once again, an ISERROR() function is nested inside an IF() function to mark any #N/A errors as blanks.

Game statistics:
=IF(ISERROR(INDEX(NBA_1920[GP],MATCH([@Player],NBA_1920[PLAYER],0))),"",(INDEX(NBA_1920[GP],MATCH([@Player],NBA_1920[PLAYER],0))))
Draft year:
=IF(ISERROR(INDEX(NBA_Draft_Dates[Draft Year],MATCH([@Player],NBA_Draft_Dates[Player],0))),"",(INDEX(NBA_Draft_Dates[Draft Year],MATCH([@Player],NBA_Draft_Dates[Player],0))))

In order to list years experience (after the 2019-2020 season), a new column in the main table is created called "NBA Experience." IF() functions are used to return "Undrafted" if the Draft Year column says "Undrafted," a blank if the Draft Year column has a blank, or else the value in Draft Year is subtracted from 2020 to give the number of years of NBA experience by the end of the 2019-2020 season. Conditional Formatting is then used to highlight players without filled in Draft Year (that have either blanks or “Undrafted” listed instead) in red fill.

Draft year data compiled from the best results of my research (information was from ESPN's individual player pages and player Wikipedia pages if the player was not listed on ESPN) is manually added to fill in any flagged cells.

In order to do analysis between leagues, positions are grouped together since both leagues name their positions slightly differently:
            Guard includes the guard and guard-forward positions in the WNBA, and the guard, shooting guard, and point guard positions in the NBA.
            Center includes the center and center-forward positions in the WNBA, and the center position in the NBA.

Averages are found for each grouped position using multiple SUMIFS() divided by multiple COUNTIFS()

=(SUMIFS(WNBA_Vets[2020],WNBA_Vets[2020],"<>Didn't play",WNBA_Vets[Position],"G")+SUMIFS(WNBA_Vets[2020],WNBA_Vets[2020],"<>Didn't play",WNBA_Vets[Position],"G-F"))/(COUNTIFS(WNBA_Vets[2020],"<>Didn't Play",WNBA_Vets[Position],"G")+COUNTIFS(WNBA_Vets[2020],"<>Didn't Play",WNBA_Vets[Position],"G-F"))

Explanation for code:
            SUMIFS: sum salary IF (salary has a value AND has "G" position) + sum salary IF (salary has a value AND has "G-F" position)
            COUNTIFS: count IF (salary has a value AND has "G" position) + count IF (salary has a value AND has a "G-F" position)
            Average = Sum/Count

NOTE: The amounts can be checked by filtering by position, then using the AGGREGATE() function, but this is not dynamic and not preferred for the main method.

2. Average and Median Salaries For Players Within the Top 25% of Points Per Game:

In order to find the top 25% of players in Points Per Game, the 3rd quartile threshold point is found using the QUARTILE.INC() function for both leagues.

=QUARTILE.INC(WNBA_Final[Points Per Game],3)

Using the quartile calculation, average and median salaries for all players above the threshold point are calculated using the AVERAGEIF() function for the average, and the MEDIAN() function paired with an IF() statement in an array formula for the median.

=AVERAGEIF(WNBA_Final[Points Per Game],">" & $C$55,WNBA_Final[2020])

Explanation for code:
IF Points Per Game for a player is greater than the 3rd quartile threshold (cell C55 in this case), THEN take the average of the 2020 salaries. Using AVERAGEIF() with a value from another cell and a logical operator in the criteria argument requires concatenation with the & character.

{=MEDIAN(IF(WNBA_Final[Points Per Game]>$C$55,WNBA_Final[2020]))}

Explanation for code:
IF Points Per Game for a player is greater than the 3rd quartile threshold (cell C55 in this case), THEN take the median of the 2020 salaries.

The upper 25% of Points Per Game represents players that, on average, score more than 14.45 points per game in the WNBA and more than 13.2 points per game in the NBA.

Discussion

This project examined the differences in salary amounts paid to the players across the WNBA and the NBA. Average, median, and maximum salary amounts were calculated for the overall leagues, charts were made to compare the two leagues, and then further comparison was done for subgroups within: veterans, rookies, those players who were within the Top 25% of Points Per Game, players within the Top 25% of Minutes Per Game, and players within the Top 25% of Games Played. Within the veterans group, WNBA and NBA salaries were also compared across similar years experience levels and across similar player positions. Within the rookies group, comparisons were also done across similar player positions.

No matter which subgroup was looked at, there was a clear discrepancy between the salaries that players within the WNBA and NBA receive—the NBA players received a great magnitude of larger salaries than their WNBA counterparts did. For rookies, the maximum salary in the NBA was $9,757,440.00 in 2020-2021, compared to the WNBA's maximum salary of $70,040.00 in 2021—a difference of $9,687,400. The median salary for rookies in the NBA was $1,988,500.00 in 2020-21, versus the WNBA's $61,543.00 in 2021—a gap of $1,926,957. For veterans and within the leagues overall, the maximum salary in the NBA was $43,006,362.00 in 2020-2021 and only $221,450.00 in 2021 for the WNBA—for a whopping $42,784,912.00 difference. The median salary for veterans was $7,316,666.50 in 2020-21 in the NBA and $167,500.00 in 2021 in the WNBA—a gap of $7,149,166.50. The median salary in the entire NBA was $3,095,432.00 in 2020-2021 and $110,000.00 in 2021 in the entire WNBA—a disparity of $2,985,432.00.

Looking at comparative attributes of players (similar years experience, position played, and being in the top 25% of the league in Points Per Game, Minutes Per Game, and Games Played) across the two leagues, this disparity in pay continues. For rookies, players in the WNBA had a median of about $60,000 in 2021 no matter the position, whereas the NBA players had a median of about $1,000,000 to $2,000,000 in 2020-2021. For veterans, similar years of experience led to major gaps between the leagues—for example, veterans who played 5 years professionally had a median salary difference of $4,559,516 during the WNBA 2021/NBA 2020-2021 seasons. WNBA veteran athletes had a median salary of around $170,000 for each position in the 2021 season, compared to the NBA veteran atheletes that had around $5,000,000 to $7,000,000 for each position during the 2020-2021 season. For the players in the entirety of the leagues within the top 25% of Points Per Game during the WNBA 2020/NBA 2020-2021 seasons, there was a gap in median salary of about $4,808,000; for players with similar scoring output, the women are paid far less than their male counterparts for contributing the same percentage of points within their own league. Similarly, for players who played comparatively the same number of minutes and were within the top 25% of Minutes Per Game, there was a gap of median salary of about $4,502,840 for the WNBA 2021/NBA 2020-2021 seasons—NBA players are rewarded far more than the WNBA players are for similar minutes of playing time per game. Finally, for players within the top 25% of Games Played during the WNBA 2020/NBA 2020-2021 season, the difference in median salary was about $3,204,200—despite being just as important players as and having to play a similar high percentage of games as the NBA players do, the WNBA players continue to be paid much less.

The gender wage gap is not a phenomenon new or unique to professional basketball. According to the United States Census Bureau, in 2020, the median female-to-male earnings ratio was 0.830—women earned 83 cents to every dollar made by men. Furthermore, the wage gap widens when also accounting for race. Black and African American women—a large makeup of the league—have lower median incomes than non-Hispanic white women do. Despite putting in the same amount of effort and work, and having comparable contributions to their respective teams, professional women basketball athletes receive only a small percentage of what the men do. Veteran players have the most experience and tend to contribute the most to their teams. However, in the WNBA 2021/NBA 2020-2021 season, the highest paid WNBA veteran received a salary ($221,450) about 44 times less than the highest paid NBA rookie did ($9,757,440). The median salary for WNBA veterans ($167,500) was about 12 times less than what the median salary for NBA rookies ($1,988,500) was in the same seasons. To be a professional athlete means to be an incredibly skilled and high level performer in your sport. Add in the physicality and toll on one's body basketball does, the wages that the WNBA players receive are unaccetable.

While there are numerous reasons and hurdles behind the lack of popularity towards women's sports, it is reasonable to acknowledge there most likely will never be the same amount of interest and financial success in the WNBA as there is in the NBA. To expect there to be complete parity in salaries and have the women making tens of millions of dollars is unrealistic. However, there can be improvements made to support the best women's basketball athletes and lessen these major salary disparities. There are already new contract deals that were signed to steadily increase salary amounts over time, increase the number of games shown on major TV networks, and provide bonuses and benefits to the players (including better travel accomodations and improved family planning options). However, if these salary amounts are not properly addressed going forward, we will continue to see the pattern we already have—WNBA players will spend their off-seasons on international teams, risking injury and sometimes personal safety, to sometimes make more than a million dollars.

While the WNBA will most likely never reach the height of popularity the NBA enjoys and, therefore, will not have players receiving salaries as high as the NBA, these professional athletes deserve to be paid an appropriate rate to reflect their skills and hard work. Women's sports need to be supported and to do that, there needs to be increased positive media exposure and negotiations to increase pay so that some of the best athletes in the world are fairly compensated.
